# What is OwnerShip

+ 所有権とは、Rustの中心的な機能である

## 所有権規則

+ Rustの各値は、`所有者`と呼ばれる変数と対応している
+ いかなる時も`所有者`はひとつである
+ `所有者`がスコープから外れたら、値は破棄される

## 変数スコープ

+ 変数スコープとは、要素が有効になるプログラム内の範囲のこと
+ 例としては以下のような形

```rust
#fn main() {
{                              // 変数sは有効になっていない
        let s = "hello";       // 変数sはここから有効になる
        println!("s = {}",s);  // 変数sで作業
}                              // 変数sは無効
#}
```
